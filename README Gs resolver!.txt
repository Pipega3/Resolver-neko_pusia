Ru

Resolver Mode: выбор приоритета фикса анти-аима (Adaptive, Brute Force).
Bone Priority: приоритет костей для стрельбы (Head, Neck, Stomach).
Brute Steps: улучшает фикс анти-аима по фазам (3-8, стандартно 3-7).
Enable Crouch Prediction Fix: улучшает back-shot при crouch static.
Fix Crouch Jitter: улучшает back-shot и вычисляет crouch jitter.
MultiPoint Scale %: повышает точность MultiPoint (лучше 70-90%).
Enable Spread Hitchance: включает ускорение выстрела (связано с Fire start ms + Hitchance spread %).
Fire start ms + Hitchance spread %: ускоряет выстрел по hitchance (адаптировано из Neverlose).
Smooth Stop: плавный авто-стоп (не конфликтует с auto-strafe).
Stop Delay (ms): скорость авто-стопа (лучше -100 до -50).
LagComp Ticks: коррекция delay на тиках (от -10-5 до 20).
Only LagComp on Ground: учитывает LagComp только на земле.
Max Extrapolate Ticks: точность определения позиции врага (лучше 5-10, до 25).

Eng


Resolver Mode: anti-aim fix priority (Adaptive, Brute Force).
Bone Priority: aimbot bone priority (Head, Neck, Stomach).
Brute Steps: improves anti-aim fix phases (3-8, default 3-7).
Enable Crouch Prediction Fix: better back-shot detection on crouch static.
Fix Crouch Jitter: improves back-shot and detects crouch jitter.
MultiPoint Scale %: increases MultiPoint accuracy (70-90% recommended).
Enable Spread Hitchance: enables shot acceleration (linked with Fire start ms + Hitchance spread %).
Fire start ms + Hitchance spread %: faster hitchance-based shooting (improved from Neverlose).
Smooth Stop: smooth auto-stop (no conflict with auto-strafe).
Stop Delay (ms): auto-stop speed (best: -100 to -50).
LagComp Ticks: tick delay adjustment (from -10-5 to 20).
Only LagComp on Ground: applies LagComp only when grounded.
Max Extrapolate Ticks: enemy position accuracy (5-10 recommended, up to 25).
